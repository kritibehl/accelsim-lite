# Benchmark Runner

AccelSim-Lite runs benchmark-style validation across workload latency, throughput, memory pressure, scheduler behavior, and runtime regression thresholds.

## Covered signals

- p95 latency drift
- throughput regression
- scheduler contention
- memory-pressure behavior
- workload trace comparison
- build-profile validation

## Scope

Educational C++ runtime/performance validation workflow. Not hardware benchmarking.
