# Limitations and Claims Boundary

## What AccelSim-Lite is

AccelSim-Lite is an educational C++ runtime/performance experimentation platform.

It models:
- workload latency and throughput
- bottleneck shifts
- queue pressure
- memory-port sensitivity
- runtime execution-engine concepts
- validation and regression behavior

## What it is not

AccelSim-Lite does not claim to be:
- a cycle-accurate CPU/GPU simulator
- a real accelerator benchmark
- a production LLM inference system
- a CUDA/NCCL benchmark
- a V8 implementation
- a JavaScript engine
- a WebAssembly runtime
- a production compiler/JIT validation harness

## Why this boundary matters

The project is designed to show systems reasoning, performance methodology, C++ implementation discipline, validation workflows, and reproducible reporting.

The metrics are simulator-derived and should be interpreted as controlled experimental outputs, not hardware measurements.
